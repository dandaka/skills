# Skills Repo

This is a **public repository**. Before committing anything, always check for personal or security-sensitive information:

- Hardcoded IDs (account IDs, zone IDs, resource IDs)
- API keys, tokens, secrets, passwords
- Personal email addresses, phone numbers
- Internal server IPs or hostnames
- Personal domain names or project names that shouldn't be public

Replace any such values with placeholders (e.g. `<ACCOUNT_ID>`, `<ZONE_ID>`, `<API_KEY>`).
